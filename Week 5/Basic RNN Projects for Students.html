Basic RNN Projects for Students

1. Text Generation with Basic RNN

Overview: Train a basic RNN model to generate text character by character using simple recurrence and Backpropagation Through Time (BPTT).

Dataset: Use the Shakespeare Texts Dataset from Kaggle.

Learning Objective: Understand sequence modeling using a basic RNN. Learn how to preprocess text data and set up RNNs for simple text generation tasks.

Resources:

  • Text Generation with Basic RNN Tutorial
  • Char-RNN Implementation on GitHub

2. Simple Time Series Prediction

Overview: Apply a basic RNN to predict future values in a numeric time series, such as temperature or stock prices.

Dataset: Use the Stock Prices Dataset or Weather Dataset from Kaggle.

Learning Objective: Practice applying RNNs to numeric time series data with Backpropagation Through Time (BPTT) for training and forecasting future values.

Resources:

  • Simple Time Series Prediction in PyTorch
  • Kaggle's Time Series Analysis Course

3. Sentiment Analysis with Basic RNN

Overview: Build a basic RNN model to classify the sentiment (positive/negative) of text reviews using simple text recurrence and BPTT.

Dataset: Use the IMDB Movie Review Dataset from Kaggle.

Learning Objective: Explore sentiment analysis using basic RNNs for text classification tasks and practice how to implement BPTT for gradient computation.

Resources:

  • Sentiment Analysis with PyTorch
  • Word2Vec NLP Tutorial on Kaggle

These projects are designed to help students practice the implementation of basic RNNs and Backpropagation Through Time (BPTT) techniques in simple real-world applications. For further learning, visit Goodfellow's Deep Learning Book and PyTorch Resources.
