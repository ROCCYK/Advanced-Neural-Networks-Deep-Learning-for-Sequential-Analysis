Comprehensive Guide to Tensor Creation in PyTorch

This page covers various methods for creating tensors in PyTorch, including creating tensors from data, using random values, generating special tensors, and specifying data types.

1. Tensor Creation from Data

import torch

# Creating a tensor from data (e.g., a list)
tensor_data = torch.tensor([1.0, 2.0, 3.0])
print(tensor_data)

# Creating a tensor from a NumPy array
import numpy as np
numpy_array = np.array([4, 5, 6])
tensor_from_numpy = torch.from_numpy(numpy_array)
print(tensor_from_numpy)

# Creating a tensor with a specified dtype
tensor_with_dtype = torch.tensor([1.0, 2.0, 3.0], dtype=torch.float32)
print(tensor_with_dtype)

2. Basic Tensor Creation

# Create an uninitialized tensor
tensor_empty = torch.empty(2, 3)
print(tensor_empty)

# Create a tensor filled with zeros
tensor_zeros = torch.zeros(3, 3)
print(tensor_zeros)

# Create a tensor filled with ones
tensor_ones = torch.ones(2, 2)
print(tensor_ones)

# Create a tensor filled with a specific value
tensor_full = torch.full((2, 3), 7)
print(tensor_full)

3. Creating Tensors with Ranges

# Create a 1D tensor with values from start to end with a step size
tensor_arange = torch.arange(0, 10, 2)
print(tensor_arange)

# Create a 1D tensor with evenly spaced values between start and end
tensor_linspace = torch.linspace(0, 1, steps=5)
print(tensor_linspace)

4. Special Tensors

# Create an identity matrix of size n x n
tensor_eye = torch.eye(4)
print(tensor_eye)

5. Random Tensor Creation

# Create a tensor filled with random numbers from a uniform distribution between [0, 1)
tensor_rand = torch.rand(2, 3)
print(tensor_rand)

# Create a tensor filled with random numbers from a normal distribution (mean 0, variance 1)
tensor_randn = torch.randn(2, 3)
print(tensor_randn)

# Create a tensor filled with random integers between low and high
tensor_randint = torch.randint(0, 10, (3, 3))
print(tensor_randint)

# Return a 1D tensor with a random permutation of integers from 0 to n-1
tensor_randperm = torch.randperm(5)
print(tensor_randperm)

6. Creating Tensors with Specific Data Types

# Create a double precision (torch.float64) tensor
tensor_double = torch.DoubleTensor([1.0, 2.0, 3.0])
print(tensor_double)

# Create a single precision (torch.float32) tensor
tensor_float = torch.FloatTensor([1.0, 2.0, 3.0])
print(tensor_float)

# Create an integer (torch.int32) tensor
tensor_int = torch.IntTensor([1, 2, 3])
print(tensor_int)

# Create a tensor on a GPU device (if available)
if torch.cuda.is_available():
    tensor_cuda = torch.cuda.FloatTensor([1.0, 2.0, 3.0])
    print(tensor_cuda)

7. Converting Data to Tensors

# Convert a NumPy array to a tensor
numpy_array = np.array([7, 8, 9])
tensor_from_numpy = torch.from_numpy(numpy_array)
print(tensor_from_numpy)

# Convert data into a tensor without copying if possible
tensor_as_tensor = torch.as_tensor([1, 2, 3])
print(tensor_as_tensor)

# Check the dtype of a tensor
print(tensor_as_tensor.dtype)

8. Creating Tensors Like Other Tensors

# Create a tensor with the same size as another tensor but filled with zeros
tensor_zeros_like = torch.zeros_like(tensor_from_numpy)
print(tensor_zeros_like)

# Create a tensor with the same size as another tensor but filled with ones
tensor_ones_like = torch.ones_like(tensor_from_numpy)
print(tensor_ones_like)

# Create an uninitialized tensor with the same size as another tensor
tensor_empty_like = torch.empty_like(tensor_from_numpy)
print(tensor_empty_like)

Summary

This guide covered various methods for creating tensors in PyTorch, including creating tensors from data, using random values, generating special tensors, and specifying data types. Understanding these commands is crucial for efficiently working with tensors in your deep learning projects.
