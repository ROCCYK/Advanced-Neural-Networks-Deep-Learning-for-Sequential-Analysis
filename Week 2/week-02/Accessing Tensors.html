Comprehensive Guide to Tensor Operations in PyTorch

This page covers various tensor operations in PyTorch, including arithmetic operations, matrix operations, comparison operations, mathematical functions, advanced operations like autograd, broadcasting, slicing, and indexing.

1. Basic Arithmetic Operations

  • Addition: torch.add(tensor1, tensor2) or tensor1 + tensor2
import torch
a = torch.tensor([1, 2, 3])
b = torch.tensor([4, 5, 6])
result = torch.add(a, b)  # or result = a + b
print(result)  # Output: tensor([5, 7, 9])
  • Subtraction: torch.sub(tensor1, tensor2) or tensor1 - tensor2
result = torch.sub(a, b)  # or result = a - b
print(result)  # Output: tensor([-3, -3, -3])
  • Multiplication: torch.mul(tensor1, tensor2) or tensor1 * tensor2
result = torch.mul(a, b)  # or result = a * b
print(result)  # Output: tensor([4, 10, 18])
  • Division: torch.div(tensor1, tensor2) or tensor1 / tensor2
result = torch.div(a, b)  # or result = a / b
print(result)  # Output: tensor([0.2500, 0.4000, 0.5000])

2. Matrix Operations

  • Matrix Multiplication: torch.matmul(tensor1, tensor2) or tensor1 @ tensor2
a = torch.tensor([[1, 2], [3, 4]])
b = torch.tensor([[5, 6], [7, 8]])
result = torch.matmul(a, b)  # or result = a @ b
print(result)  # Output: tensor([[19, 22], [43, 50]])
  • Element-wise Multiplication: torch.mul(tensor1, tensor2)
result = torch.mul(a, b)
print(result)  # Output: tensor([[ 5, 12], [21, 32]])
  • Transpose: tensor.T or torch.transpose(tensor, dim0, dim1)
result = a.T  # or result = torch.transpose(a, 0, 1)
print(result)  # Output: tensor([[1, 3], [2, 4]])
  • Matrix Inverse: torch.inverse(tensor) (for square matrices)
a = torch.tensor([[1.0, 2.0], [3.0, 4.0]])
result = torch.inverse(a)
print(result)  # Output: tensor([[-2.0000,  1.0000], [ 1.5000, -0.5000]])
  • Matrix Determinant: torch.det(tensor)
result = torch.det(a)
print(result)  # Output: tensor(-2.0000)

3. Reduction Operations

  • Sum: torch.sum(tensor)
result = torch.sum(a)
print(result)  # Output: tensor(10.)
  • Mean: torch.mean(tensor)
result = torch.mean(a)
print(result)  # Output: tensor(2.5000)
  • Max: torch.max(tensor)
result = torch.max(a)
print(result)  # Output: tensor(4.)

4. Comparison Operations

  • Greater than: tensor1 > tensor2
result = a > 2
print(result)  # Output: tensor([[False, False], [ True,  True]])
  • Less than: tensor1 < tensor2
result = a < 3
print(result)  # Output: tensor([[ True,  True], [False, False]])
  • Equal: torch.eq(tensor1, tensor2)
result = torch.eq(a, b)
print(result)  # Output: tensor([[False, False], [False, False]])

5. Mathematical Functions

  • Exponential: torch.exp(tensor)
result = torch.exp(a)
print(result)  # Output: tensor([[ 2.7183,  7.3891], [20.0855, 54.5981]])
  • Logarithm: torch.log(tensor)
result = torch.log(a)
print(result)  # Output: tensor([[0.0000, 0.6931], [1.0986, 1.3863]])
  • Square Root: torch.sqrt(tensor)
result = torch.sqrt(a)
print(result)  # Output: tensor([[1.0000, 1.4142], [1.7321, 2.0000]])

6. Rounding Operations

  • Round: torch.round(tensor)
result = torch.round(torch.tensor([0.5, 1.3, 2.7]))
print(result)  # Output: tensor([0., 1., 3.])
  • Floor: torch.floor(tensor)
result = torch.floor(torch.tensor([0.5, 1.3, 2.7]))
print(result)  # Output: tensor([0., 1., 2.])

7. Norms

  • L2 Norm: torch.norm(tensor) (Euclidean norm)
result = torch.norm(a)
print(result)  # Output: tensor(5.4772)

8. Broadcasting

a = torch.tensor([1, 2, 3])
b = torch.tensor([[4], [5], [6]])
result = a + b  # Broadcasting happens automatically
print(result)  
# Output: 
# tensor([[5, 6, 7],
#         [6, 7, 8],
#         [7, 8, 9]])

9. Advanced Operations: Autograd

Autograd is PyTorch's automatic differentiation engine that powers neural network training. It enables the computation of gradients automatically.

import torch

# Create a tensor with requires_grad=True to track computations
x = torch.tensor([2.0, 3.0], requires_grad=True)

# Perform operations on the tensor
y = x * 2
z = y.mean()

# Compute gradients
z.backward()

# Gradients
print(x.grad)  # Output: tensor([1., 1.])

10. Slicing and Indexing

  • Basic Indexing: Access elements using indices.
a = torch.tensor([1, 2, 3, 4, 5])
result = a[1]  # Access the second element
print(result)  # Output: tensor(2)
  • Slicing: Use the slice notation to extract sub-tensors.
result = a[1:4]  # Extract elements from index 1 to 3
print(result)  # Output: tensor([2, 3, 4])
